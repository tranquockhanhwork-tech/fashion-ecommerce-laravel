# AI Product Image Prompts

Môi trường hiện tại chưa có công cụ generate ảnh AI trực tiếp, nên file này lưu sẵn prompt để tạo ảnh sản phẩm nhất quán khi dùng Midjourney, ChatGPT Images, Flux, Ideogram hoặc công cụ tương tự.

## Premium Streetwear Set

Prompt:

```text
Luxury streetwear fashion product shoot, premium layered outfit, black and mocha gold palette, oversized jacket with tailored inner set, full-body model standing in a clean editorial studio, soft premium lighting, neutral warm background, high-detail fabric texture, realistic folds, modern Vietnamese fashion e-commerce style, front-facing product focus, 4k, no text, no watermark
```

Biến thể ảnh nên tạo:

```text
1. Front view, centered composition, clean e-commerce studio.
2. Three-quarter side pose, showing layering and silhouette.
3. Close-up crop on fabric texture, collar, sleeves and premium finishing.
```

## Generic Catalog Template

```text
Premium fashion e-commerce product photo, [PRODUCT_NAME], [COLOR], clean studio background, soft luxury lighting, realistic fabric detail, centered composition, front view, high sharpness, premium editorial look, no accessories unless product requires it, no text, no watermark, 4k
```

## Consistency Notes

- Ưu tiên nền sáng trung tính hoặc kem ấm.
- Ánh sáng mềm, có chiều sâu, tránh bóng gắt.
- Giữ tỉ lệ khung dọc 3:4 để khớp card sản phẩm hiện tại.
- Không thêm chữ, logo giả, watermark hoặc background quá phức tạp.
